# Google Drive API サンプルコード

このサンプルでは、Google ドライブへのファイル保存、共有ドライブへのファイルの保存についてのコードを記載しています。<br/>
Google ドライブは Google_Service_Drive サービスを使用して操作します。

## ファイル構成
```
トップ
│
├ README.md .. このファイル
│
└ DriveApiController.php .. サンプルコード
```
